1) Actualizar o sistema updata -> upgrade
2.1) Instalar nodejs: sudo apt install nodejs
2.2) Instalar npm: sudo apt install npm
2.3) Actualizar npm con sudo npm install npm@latest -g
3.1) Configurar npm: sudo npm config set prefix /usr/local/
3.2) Instalar socket.io para poder traballar con webSockets: sudo npm install -g socket.io@2.0.4
3.3) Instalar tail pra leer ficheiros: sudo npm install tail
3.4) Instalar hightCharts: sudo npm install: sudo npm install --save highcharts
